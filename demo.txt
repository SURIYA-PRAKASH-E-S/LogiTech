LogiTech Route Flow - Demo Project

This is a logistics management system with real-time tracking and route optimization.

Features:
- Real-time driver location tracking
- Route optimization and planning
- Image upload for delivery proof
- Payment integration with Razorpay
- Firebase real-time database
- Modern React UI with Tailwind CSS

Setup Instructions:
1. Install dependencies: npm install
2. Copy .env.example to .env and fill in your API keys
3. Start development server: npm run dev
4. Start backend server: npm run server:dev
5. For full development: npm run dev:full

Build & Deploy:
- Build: npm run build
- Deploy to Firebase Hosting: npm run firebase:deploy:hosting

Technologies Used:
- React 18 with TypeScript
- Vite for build tooling
- Firebase for backend services
- Tailwind CSS with shadcn/ui
- Leaflet for mapping
- Razorpay for payments
